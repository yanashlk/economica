# Brief

Вебзастосунок для збору відповідей через динамічну форму  
та адміністрування відповідей через адмін-панель.

## Стек
- Frontend: Vue 3 + Vite
- Backend: Node.js (Vercel Serverless)
- Database: PostgreSQL (Neon)

## Структура
- `web/` — фронтенд
- `api/` — бекенд (API)
- `db/` — SQL-скрипти
